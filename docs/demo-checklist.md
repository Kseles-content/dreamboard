## 2) Создание доски

### 2.2 Доска из шаблона
- [ ] Выбрать template

## 4) Поиск и фильтрация досок
- [ ] Поиск по `query`
- [ ] Фильтр `Pinned only`
- [ ] Фильтр `Updated last 7 days`
- [ ] Проверить корректность результатов

## 5) Resume flow
- [ ] Открыть доску
- [ ] Вернуться на dashboard
- [ ] Проверить блок "Continue where you left off"
