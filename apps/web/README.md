# DreamBoard Web — Editor core v1 (Week 5)

Stage 3 focus: text-card editor core with real API (no mocks).

## Features
- Login
- Boards list
- Open board
- Add text card
- Edit text card
- Delete text card
- Persisted via API endpoints
- Loading/error/empty states

## Run
```bash
cd apps/web
npm ci
npm run dev
```
Open http://localhost:3100

API default: `http://localhost:3000` (editable on login screen).

## Build/Test
```bash
cd apps/web
npm run build
npm run test
```
